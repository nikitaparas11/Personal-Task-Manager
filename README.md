# 📝 Personal Task Manager

A simple full-stack task management application built using HTML, CSS, JavaScript, Node.js, and Express.js.

## Features

- ➕ Add new tasks
- 📋 View all tasks
- ✔ Mark tasks as complete or incomplete
- ✏ Edit existing tasks
- 🗑 Delete tasks
- 🔍 Search tasks by title
- 📊 View active and completed task counts
- ⏰ Highlight overdue tasks
- 💾 Save tasks in a JSON file so they remain after server restarts

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Storage
- JSON File (`tasks.json`)

## How to Run
   website link - https://personal-task-manager-git-main-nikita-paras-projects.vercel.app/
1. Install dependencies

```bash
npm install
```

2. Start the server

```bash
npm run dev
```

or

```bash
node server.js
```

3. Open `index.html` using Live Server.
